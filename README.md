# dmt-bicing

The data has been extracted from opendata barcelona
there are three files
the neighborhoods poligons from the city of barcelona
 the bicing stations info such as location and name 

and the stations status how many bikes available are and so on:
https://opendata-ajuntament.barcelona.cat/data/ca/dataset/estat-estacions-bicing

in order to generate the data of locations we have selected only the ones that appeared in the last document (the most updated one)

In order to push it to github we have add it monthly (only months available)

we have also reduced the data to have a 15 min granularity from 6 to 22 (hours where people tend to use more the bike)
